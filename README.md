# metric.AI 📊🛒

> Aplicação robusta de integração entre plataformas de E-Commerce, focada em consolidação de métricas e automação de processos.

## 🚀 Sobre o Projeto
O **metric.AI** nasce da necessidade de centralizar e analisar dados de vendas e estoque de diferentes marketplaces. Atualmente, o sistema conta com uma integração ativa com a API do **Mercado Livre (MELI)**, permitindo sincronização em tempo real e gerenciamento seguro de dados.

O projeto conta com um ecossistema completo: um ecossistema **Back-end robusto em Spring Boot** e uma interface **Front-end interativa**.

---

## 🛠️ Tecnologias e Ferramentas

### Back-end
* **Java & Spring Boot** (Core da aplicação)
* **Spring Security & JWT (JSON Web Tokens)**: Sistema de autenticação completo e seguro para proteção das rotas da API.
* **IntelliJ HTTP Client (`test-api.http`)**: Utilizado para testes ágeis e automatizados de requisições HTTP diretamente na IDE.
* **Ngrok**: Ferramenta de tunelamento para simulação e recebimento de Webhooks locais das APIs de e-commerce.

### Front-end
* **JavaScript / HTML / CSS** (Interface do usuário integrada)
